Usage Workflow

Discovery Mode vs. Monitoring Mode

  1. Toggle ON "Nearby Wells" → Gray teardrops appear in a 5x5 section grid around your properties
  2. Browse and explore → Click on gray markers to see well details
  3. Track interesting wells → Click "Track This Well" in popup to add to your tracked wells
  4. Toggle OFF → Return to clean monitoring view showing only your purple tracked wells

Key Benefit: The size difference (60% vs 80%) ensures tracked wells remain visually dominant while nearby wells provide context without cluttering the map.
